------- YandexAudio API
YandexAudio API — это JavaScript-библиотека для воспроизведения аудио в браузере. 
С ее помощью на странице можно проигрывать музыку, аудиокниги и прочие звуковые файлы. 
С примером использования данной библиотеки можно ознакомиться на данной странице.

https://yandex.ru/dev/audio/jsapi/doc/dg/concepts/webaudio-usage.html
